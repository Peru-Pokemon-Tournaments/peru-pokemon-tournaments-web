# Peru Pokemon Tournaments Web

Web to manage players' inscriptions and tournaments

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
